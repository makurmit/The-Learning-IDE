Chapter 4: Methods
Cheat Sheet

Methods
	public/private returnType name(type var)
	{
		...
		return var;
	}
	public - accessible outside of its class
	private - accessible only to its class
	return types - any primitive type or object, "void" means it returns nothing
	name - user given, typically camalCased
	() - parameters, anything needed inside the method, must give type and name for variable. This can be empty
	
	Python:
		def name(var):
			return var #not necessary
	
	Ruby:
		def name(var)
			return var
		end
